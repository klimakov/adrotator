# Тестирование AdRotator

## Развёрнутые сервисы

После `docker compose up -d` доступны:

| URL | Описание |
|-----|----------|
| http://127.0.0.1/ | Админка (дашборд, кампании, креативы, площадки) |
| http://127.0.0.1/api/health | Проверка работы API |
| http://127.0.0.1/sdk/ad.js | Web SDK для встраивания |
| http://127.0.0.1/sdk/test.html | **Тестовая страница с баннером** — открой в браузере |

## Проверка в браузере

1. **Админка**  
   Открой http://127.0.0.1/ — должен загрузиться дашборд с блоками «Активных кампаний», «Показов сегодня» и т.д.

2. **Тест баннера**  
   Открой http://127.0.0.1/sdk/test.html — на странице должен отобразиться баннер (placeholder 300×250), если перед этим был выполнен сценарий создания кампании/креатива/площадки (см. ниже).

## Автоматический API-тест

Создаёт кампанию, креатив, площадку, привязывает креатив и проверяет выдачу баннера:

```bash
./e2e/api-test.sh http://127.0.0.1/api
```

Перед проверкой тестовой страницы в браузере запусти этот скрипт, чтобы в зоне `e2e-zone` был баннер.
